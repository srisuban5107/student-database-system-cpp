# 📚 Student Database System (C++)

A **console-based Student Database Management System** developed using **C++**, designed to store and manage **real-life school student records** in a **human-readable format**.

This project is ideal for:
- 🎓 Academic projects
- 💼 Portfolio showcase
- 🎤 Interview explanations

---

## ✨ Features

- Add student records with validation
- View all students
- Search student by ID
- Update selected student fields (menu-driven)
- Delete student records
- Prevents duplicate **ID** and **Name**
- Case-insensitive input (auto converts to UPPERCASE)
- Phone number is **mandatory** (10 digits)
- Address is **mandatory**
- Email is optional
- Age and DOB verification using system date
- Supports subjects with **sub-parts**
  - Example: `BIOLOGY → BOTANY + ZOOLOGY`
- Group option shown **only for Class 11 & 12**
- Saves data to a **clean, readable `.txt` file**

---

## 🛠 Technologies Used

- Language: **C++**
- Concepts:
  - File Handling (`fstream`)
  - Structures and Vectors
  - Input Validation
  - String Manipulation
  - Date Validation (`ctime`)
